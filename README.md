# \# GameStore — Интернет-магазин игровых товаров на WordPress + WooCommerce

🔗 **Живая демо-версия сайта:** http://kravchrt.beget.tech

# 

# Разработка кастомной темы для игрового маркетплейса с интеграцией e-commerce функционала, адаптивной версткой и оптимизацией под современные требования веб-стандартов.

# 

# \## 🛠 Технологический стек (Stack)

# \* \*\*CMS:\*\* WordPress 6.x

# \* \*\*E-Commerce:\*\* WooCommerce (архитектура магазина, фильтры, корзина)

# \* \*\*Frontend:\*\* HTML5, CSS3 (BEM методология), JavaScript (ES6+) 

# \* \*\*Графика и дизайн:\*\* Figma (работа с макетом, экспорт ассетов), Adobe Photoshop (оптимизация графики)

# \* \*\*AI-Tools:\*\* Cursor AI (использование искусственного интеллекта для оптимизации и ускорения написания кастомных PHP-шаблонов и JS-скриптов)

# 

# \## 🚀 Реализованный функционал (Features)

# 1\. \*\*Кастомная посадка на CMS:\*\* На основе чистого HTML/CSS кода разработана независимая, легкая тема WordPress без использования тяжелых конструкторов (Elementor и др.).

# 2\. \*\*E-commerce логика:\*\* Настроена полноценная работа WooCommerce: кастомизация карточек товаров, структуры категорий, корзины и процесса оформления заказа.

# 3\. \*\*Адаптивность:\*\* Полная кроссбраузерная и адаптивная верстка (Mobile First) для корректного отображения на смартфонах и планшетах.

# 

# \## 📂 Структура репозитория (Repository Structure)

# \* `/gamestore-theme/` — исходный код разработанной кастомной темы WordPress (шаблоны PHP, стили, скрипты).

# \* `/database/` — дамп базы данных (SQL) со структурой интернет-магазина и демонстрационным контентом для быстрого развертывания проекта.
